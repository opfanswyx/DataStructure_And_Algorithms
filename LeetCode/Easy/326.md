# Power of Three
## Description
Given an integer, write a function to determine if it is a power of three.

Example 1:
```
Input: 27
Output: true
```
Example 2:
```
Input: 0
Output: false
```
Example 3:
```
Input: 9
Output: true
```
Example 4:
```
Input: 45
Output: false
```
## Solution
```c
bool isPowerOfThree(int n) {
    if(n==0)
        return false;
    while(n!=1)
    {
        if(n%3==0)
            n/=3;
        else
            return false;
    }
    return true;
}
```
