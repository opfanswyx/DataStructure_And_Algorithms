# 算法复杂度
## 对数阶O(logn)
```c
int number=1;
while(number<n){
number=number*2;
...
}
```
## 平方阶O(n^2)
```c
for(int i=0;i<n;i++){   
      for(int j=0;j<n;j++){
         ... 
      }
  }
```
